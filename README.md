# Misbah Ullah — Creative Systems Engineer Portfolio

A manga-inspired, high-contrast digital portfolio demonstrating projects and expertise in DevOps, cloud infrastructure, systems programming, cybersecurity, and digital logic.

Built with **Next.js 16**, **React 19**, **Framer Motion**, **Anime.js**, and custom **CSS**.

---

## 🎨 Design Philosophy & Aesthetic

This portfolio steps away from traditional minimalist corporate themes, adopting a premium **manga/comic-inspired retro-punk design system**:
- **High-contrast layout**: Bold ink borders, paper textures, and halftone background gradients.
- **Accents**: Striking cyber-cyan and crimson-red color pairing to emphasize technical elements.
- **Interactive Chapter Rail**: A custom side navigation indicator tracking scrolling progress interactively.
- **Micro-animations**: Dynamic hover offsets, scaling, rotation, and slide-in animations.

---

## 📂 Project Chapters

The portfolio is structured chronologically into chapters:

### Chapter 00: Opening (Intro)
A hero section with clean typography introducing Misbah Ullah, featuring a three-dimensional background accent, hero portrait, and introductory links.

### Chapter 01: Systems (Systems Philosophy)
A look into the philosophy of low-level systems programming, networking protocols, hardware design, and cloud deployments.

### Chapter 02: Projects (Case Studies)
- **CASE 01: PHANTASM**: Blue-Green Deployment Visualizer (Docker, AWS, Terraform, GitHub Actions).
- **CASE 02: EscapeQL**: Security-flavored relational database logic design.
- **CASE 03: ITNMS**: Intelligent Transport Network Management System in C++ (custom data structures).
- **CASE 04: Medical Scheduler**: Multi-client TCP socket scheduler.
- **CASE 05: Hybrid OTP Generator**: Hardware security device using Arduino and digital logic (LFSR).
- **CASE 06: Gomodoro**: Gamified Godot focus timer.

### Chapter 03: Lab Notes (Hardware Design)
A showcase of digital logic projects and circuits, linking directly to hardware lab work.

### Chapter 04: Skills
A categorized list of core skills spanning:
- **DevOps**: Docker, AWS, Terraform, GitHub Actions, Nginx.
- **Systems**: C++, C, RISC-V, CMake, Linux.
- **Security**: SQL Injection, XSS, IDOR, Incident Response.
- **Theory**: Automata, DBMS, Digital Logic Design, Algorithms.

### Chapter 05: Contact
A quick contact form and links to reach out, featuring a customized portrait avatar and primary links.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/misbah-ullah01/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

4. Build for production:
   ```bash
   npm run build
   ```
